## Notes
Приложение для закрепления заметок с возможностью: добавлять; удалять; отслеживать кол-во знаков; фильтровать заметки, применять тюмную тему. (Данные хранятся в localStorage.)
# Стек: React, CSS.
# Старт
  1. npm i
  2. npm start
